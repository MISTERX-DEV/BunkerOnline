# EN
Open-source solution for the BUNKER board game, also as an alternative to Bunker Online!" with the ability to host the game on your own hosting without restrictions (yes, I don't like that you have to pay for the full functionality >:#)

At the moment, it only works inside the terminal, and you need to download the entire repository and run python.

It is planned to create a web interface in the future.
# RU
Решение с открытым исходным кодом для настольной игры BUNKER, также в качестве альтернативы Bunker Online!" с возможностью размещения игры на вашем собственном хостинге без ограничений (да, мне не нравится, что вам приходится платить за полный функционал >:#)

На данный момент он работает только внутри терминала, и вам нужно загрузить весь репозиторий и скачанный python.

В будущем планируется создать веб-интерфейс.
